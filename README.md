# fongwell-app
基于Node和webpack的项目基本构建


## Info

```bash
- 基于TypeScript，React
- 支持ECMAScript最新特性：如异步函数，动态导入，装饰器；
- 语言编译顺序：TypeScript ==> ES7 ==> 通过babelrc转ES5（为了antd组件按需加载）
```
